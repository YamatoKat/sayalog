## 道の駅スタンプラリーを始めたきっかけ
私は、ドライブが趣味で、週末にはいつもどこかの道を走ってます。昔はバイクでしたが今は車です。
東京に住み始めて3年経過し、近辺の観光地はほぼ制覇しちゃいまして、1度行った場所にもう一度行くのは新鮮さが欠けるため、面白みがないなあと感じていた際に、偶然立ち寄った道の駅で、このスタンプラリーブックを発見し道の駅を巡ることにしました。


## 道の駅を巡る旅の計画
どこに道の駅があるのかを検索しました。すると全国にプロットできないくらいの道の駅（1160箇所）が存在してました。

この瞬間、日本全国を回るのは、将来的に達成することとして、[関東版](http://www.kanto-michinoeki.jp/%E3%82%B9%E3%82%BF%E3%83%B3%E3%83%97%E3%83%A9%E3%83%AA%E3%83%BC2019.html)に掲載されている道の駅（175箇所）だけを巡ることに決めました。

しかし、Googlemapの経路検索では複数目的地の最適ルートの結果表示はできませんし、
複数日に分けて回る必要があるため、10分ほど考えましたが、旅行ルートが見えなかったので、手元のMacに頼ることにしました。

## 道の駅を巡る旅行計画の計算を開始します
ここまでで、データを揃いましたので、道の駅を巡る旅行計画の計算をします。
計算を始めるにあたって、自分の好みの計算手段を選び必要があります。
繰り返しになりますが、面倒くさがりの私としては、思いついたその日中にサクッと道の駅旅行計画を完成させたかったので、こんなにも簡単にできるんだと驚くくらいの方法をご紹介します。

ここで、今回得たいアウトプット（ゴール）を整理します。
1ヶ月に2回旅行に行くのを5ヶ月間行うとして、10トリップで道の駅を制覇するとした場合に、
1.どの道の駅を、2.どんなルートで周ればいいか、が得たいゴールです。
つまり、1.どの道の駅、2.どんなルートと２ステップ必要です。

1.どの道の駅は、K-means
2.どんなルートは、巡回セールスマン問題

という既存のアルゴリズムでサクッと片付きそうです。
有名なアルゴリズムだとググるとコードが出てきますので、時間節約になります。
楽するためには、最初だけ少し考える必要があります。

## 計算します

今回は、アルゴリズムとかに全くこだわりはなく、結果をすぐに知りたいので、

K-meansは、[Pythonのライブラリ](https://scikit-learn.org/stable/modules/generated/sklearn.cluster.KMeans.html)を利用します。

巡回セールスマン問題(TSP)は、ググってすぐに出てきた[こちらのブログ](https://tony-mooori.blogspot.com/2016/02/tsppython.html)にあるコードを利用したいと思います。
ブログを読むと蟻コロニー最適化の方が良いそうなので、そちらのコードをコピーして、メモ帳とかに貼り付けて、tspari.pyと言う名前でデスクトップに保存してください。
巡回セールスマン問題(TSP)のアルゴリズムの比較は色んなとこで書かれているので、興味がある方は検索して見てください。

では、実際にコード書いていきます。
他の人のコードとかを利用する時は、その仕様に合わせる必要があったりするので、ちょっと面倒です。
さらに、最終的に結果をGoogleマップに表示させたいので、出力の形式とかも気を使う必要があります。

```

import pandas as pd 
import numpy as np
from sklearn.cluster import KMeans # K-means
import tspari # TSP

#CSVデータの読み込み
df = pd.read_csv('michi1206.csv', header=None)

# 何トリップ（何日）で全ての道の駅を回りたいか
trips_total = 10
# トリップ数に道の駅をグルーピング (メルセンヌツイスターの乱数の種を 10 とする)
kmeans_model = KMeans(n_clusters=trips_total, random_state=10).fit(df.iloc[:, 3:5])
# グルーピング結果ごとにラベルを付け
labels = kmeans_model.labels_
# ラベルをデータの列に追加
df['label'] = labels

# GoogleMapで表示しやすいようにソートするためだけの関数
def result_sort(lst, value):
    return [i for i, x in enumerate(lst) if x == value]

# グルーピングして付けたラベルごとに、巡回セールスマン問題（TSP）を解く
for t in range(trips_total):
    temp = df[df['label'] == t]
    name = temp[[1, 3, 4]]
    csv = temp[[3, 4]]
    csv.to_csv('csv_out.csv', index=False)

    # 他の人のコードの仕様上ファイル読み込み
    tsp = tspari.TSP(path='csv_out.csv')

    # 蟻コロニー最適化（1000匹の蟻にあらゆるルートを歩いてもらい早く回り終わった蟻のルートを採用）
    tsp.solve(n_agent=1000)

    # GoogleMapで表示しやすいようにソートするために、新しいインデックスとしてnewindexを作成
    newindex = []
    for i in range(len(tsp.result)):
        newindex += result_sort(tsp.result, i)

    name['newindex'] = newindex
    name_s = name.sort_values('newindex', ascending=True)
    name_s.to_csv('googlemap'+ str(t) + '.csv', index=False)


```

上記のコードをmichinoeki.pyとかでデスクトップに保存して、
```
python michinoeki.py
```
と実行してください。

を実行すると、googlemap0.csvからgooglemap9.csvまでのファイルが出力されると思います。

試食が豊富で、ケーキや野菜まで試食できます。色んな種類と味のピーナッツも試食でき、昼食を節約したい方にはオススメです。
食後にはブルーベリーアイスクリーム（これは有料）がオススメです。

### ３位：道の駅　保田小学校
小学校の横にあるすごく大きなプレハブで、特産品が販売されており、食べたいものが確実に見つかると思います。果物をその場で買って、外のベンチで食べました。
最近は台風被害でプレハブが損傷を受けて、特産品の販売が中止となってました。
